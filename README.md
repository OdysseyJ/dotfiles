* Install

* clone this Repo in your local home directory.
ex) $git clone https://github.com/sublime718/dotfiles.git

* changing directory -> dotfiles
ex) $cd dotfiles

* and just execute the "bootstrap.sh"
ex) $./bootstrap.sh
